# booking-system
# Система записи на занятия

Сайт для записи на индивидуальные занятия (математика, информатика, теннис).

## Возможности

- Просмотр недельного календаря занятий
- Запись с проверкой свободного времени
- Мультизапись на несколько недель
- Опросы для каждого направления
- Личная статистика (часы за месяц)
- Напоминания о ближайших занятиях
- Админ-панель (подтверждение записей, просмотр опросов)

## Технологии

- Frontend: HTML, CSS, JS
- Backend: Node.js + Express
- БД: PostgreSQL
- Аутентификация: JWT

## Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

Особенности
При записи менее чем за 24 часа требуется подтверждение админа
Между теннисными занятиями и онлайн-уроками обязателен перерыв 1 час
Записи с одинаковым типом занятий не могут идти подряд без перерыва
Для тенниса важен адрес корта (проверяется при переезде между комплексами)
